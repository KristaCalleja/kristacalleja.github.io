Hi, I'm Krista!

I grew up by the rocky coast of Malta, where I pursued a legal career combatting illegal fisheries. In the past years, the urge to switch contexts and dive further into the tech world grew stronger. I love the creative process and the purposeful tasks programming entails.

Nowadays, alongside my legal work, I keep my tech skills up to scratch working as a ditigal nomad.

Datesy

PRODUCT DEVELOPMENT / UX / UI / FULL-STACK DESIGN / ROR / SCSS

If you're tired of looking for love, let Datesy do it for you. This app is the brainchild of a cohort of mine at the programming bootcamp. We teamed up and designed this mobile version from its very most basic features: user journey, database relationships, palettes, etc. It well received by the audience on presentation day.

A Ponte

USER JOURNEY / WIREFRAMING / UX / UI / FRONT-END / BACK-END / ROR / SCSS

A Ponte is the legal student's pocket companion. For this project I paired with a legal colleague and together we devised a platform tailored to young law students located in remote regions of LATAM. A Ponte connects them with the best sources of modern jurisprudence. With a good dose of sleek front-end design and intelligent gems on the back-end, this platform is oozing with potential.

Nuzzle

PRODUCT DEVELOPMENT / UX / UI / FRONT-END / BACK-END / ROR / SCSS

Nuzzle is a web app which connects you to your next best four-legged friends. It is a marketplace application, and it uses the four main features of a CRUD app: creating, reading, updating and deleting records. Baw-waw!

Istituto Urca

USER JOURNEY / WIRE-FRAMING / FRONT-END / HTML5 / CSS

This is a volunteer project for a local NGO working for environmental and social causes in the metropolitan area of the city of Rio. The team was looking to connect with the elderly in the neighbourhood, by recording their stories. An important element for this project was writing code with accessibility in mind.
